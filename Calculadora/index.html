Calculadora





El proyecto de la calculadora es una aplicación web que permite realizar operaciones matemáticas simples como suma, resta, multiplicación, división y potenciación, así como también calcular la raíz cuadrada de un número.

La calculadora está construida con HTML, CSS y JavaScript. El HTML define la estructura de la página, el CSS se encarga del diseño y la presentación visual de la calculadora, mientras que el JavaScript maneja la lógica y las operaciones matemáticas.

En el HTML, la calculadora consiste en un input que funciona como pantalla y una serie de botones que representan los números y las operaciones. Los botones se agrupan en divs que los contienen y los separan visualmente.

El CSS define el diseño y la presentación visual de la calculadora, incluyendo los colores de los botones, el tamaño de la pantalla y los efectos visuales cuando se presionan los botones.

El JavaScript es el encargado de manejar la lógica y las operaciones matemáticas. Se definen varias funciones que se encargan de manejar diferentes eventos, como la actualización de la pantalla, la adición de números y operaciones, y el cálculo del resultado.
